# git 사용법
1. 로컬 저장소를 생성한다.
2. 코드를 작성한다.
3. 변경내역에 대해 커밋(버전)을 생성한다.
   1. 변경사항을 git이 감지한다. (.gitignore에 등록된 파일/디렉토리는 제외)
   2. 커밋으로 남길 변경사항을 스테이징 영역에 추가한다.
   3. 메세지와 함께 커밋을 생성한다. ✌️✌️
